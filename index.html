05 — FAQ

FREQUENTLY ASKED
QUESTIONS

How does MindLang work?

+
Send voice messages to @MindLangSpeachbot. We transcribe using Whisper, analyze grammar, suggest corrections, and build vocabulary. Get instant feedback + daily study plan.

Which languages?

+
English fully supported (A1-C2). Spanish/Mandarin coming Q3 2026. Transcription works for 50+ languages.

Is voice data private?

+
Yes. Audio deleted immediately after processing. Transcripts anonymized. Never sold. Full details in Privacy Policy.

For teachers?

+
Enterprise plan includes teacher dashboards, student tracking, classroom tools. Contact enterprise@mindlang.ai

Vs ChatGPT?

+
Specialized for spoken language. Focus on pronunciation, fluency, grammar feedback, vocabulary tracking. Tuned for ESL.

Transcription accuracy?

+
98%+ for clear English. Works with accents. Background noise may reduce accuracy.

Cancel subscription?

+
Anytime from Telegram bot. Monthly no commitment. Annual refunds within 14 days.

API for developers?

+
Yes, REST API with transcription, grammar analysis, vocabulary extraction. docs.mindlang.ai

How to start?

+
Open Telegram, search @MindLangSpeachbot, send first voice. Free for 50 min/month.

Payment methods?

+
Credit cards, PayPal, crypto (BTC, ETH, USDT). Bank transfer for Enterprise.
